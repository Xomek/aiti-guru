# Aiti Guru (Тестовое задание)

## Технологии

- **React 19** - библиотека для построения пользовательских интерфейсов
- **TypeScript** - типизированный JavaScript
- **Vite** - быстрый сборщик проектов
- **React Router DOM v7** - маршрутизация
- **Axios** - HTTP-клиент для работы с API
- **Zod** - валидация схем данных
- **ESLint + Prettier** - линтинг и форматирование кода
- **Husky + lint-staged** - pre-commit хуки

## 1. Перед первым запуском установите все необходимые зависимости:

```bash
npm install
```

## 2. Запуск в режиме разработки.

Для запуска проекта в режиме разработки выполните

```bash
 npm run dev
```

## 3. Сборка для production

Если нужно собрать проект для продакшена:

```bash
npm run build
```

## 4. Предпросмотр собранного проекта

После сборки можно запустить предпросмотр:

```bash
npm run preview
```
